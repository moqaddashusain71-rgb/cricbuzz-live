# PRO Cricket Live Now Overlay

## Features
- Fully visual mobile-friendly scoreboard
- Team color themes
- Stadium background overlay
- Top scorer + wickets
- Auto-refresh every 15 sec
- Live Now app compatible

## Setup
1. Replace `API_KEY` if needed
2. Host on GitHub Pages / local server
3. Open `scoreboard.html` on mobile
4. Live Now → Go Live → Screen Share → select browser tab
5. Enable mic for commentary
